# Real Estate Tokenization Frontend

Frontend application for interacting with the Real Estate Tokenization smart contracts on Mantle Network.

## Features
- Connect wallet (MetaMask)
- View available properties
- Deposit into Senior/Junior tranches
- View yield distributions
- Monitor property occupancy rates

## Setup
```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Configuration
1. Update contract addresses in `config.js` after deployment
2. Configure Mantle Network in MetaMask
3. Get testnet MNT from faucet
